Класс Пациент
- личные данные
- ID пациента 
- список диагнозов
- список аллергических реакций

  - создать
  - изменить

Класс Пациенты
- массив пациентов
- методы:

  - найти пациента по ФИО
  - найти пациента по ID
  - найти пациента и изменить данные пациента
  - удалить пациента
